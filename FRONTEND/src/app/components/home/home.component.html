Repositorio Github
La plataforma de alojamiento de Git ha proporcionado un entorno eficiente y colaborativo, permitiendo gestionar el código de manera efectiva. A través de funcionalidades de GitHub, se ha garantizado un avance transparente.
Laravel (API/JWT)
Se ha optado por la combinación de JWT (JSON Web Token) junto con Laravel como la API. JWT ofrece un método eficiente y seguro para la autenticación, permitiendo al usuario acceder a recursos de manera segura. Laravel, con su sintaxis elegante y características robustas, ha sido la elección ideal. Esta combinación ha permitido implementar una API potente y segura, proporcionando una experiencia de usuario fluida y autenticada.
Xampp
Se decidió utilizar XAMPP como la solución integral para el entorno de desarrollo local. XAMPP ofrece un conjunto completo de herramientas, incluyendo Apache como servidor web, MySQL como sistema de gestión de bases de datos, PHP como lenguaje del lado del servidor, entre otras.
Angular
Ofrece una estructura robusta y escalable que facilita la creación de aplicaciones web modernas. Con sus características como enlace bidireccional de datos, modularidad y un sólido sistema de componentes, Angular proporciona un entorno de desarrollo que favorece la creación eficiente y mantenible de interfaces de usuario interactivas
